# ReadMaster AI - Android APK

App gerado a partir do Google AI Studio para Android.

## Como gerar APK

Este repositório contém o código-fonte do app ReadMaster AI.

Para gerar o APK:
1. Extraia os arquivos do ZIP
2. Instale Node.js
3. Execute: `npm install` e `npm run build`
4. Use o Android Studio para compilar

## Tecnologias

- React/TypeScript
- Google AI/Gemini API
- Capacitor (Mobile)
